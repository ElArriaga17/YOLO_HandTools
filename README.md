# YOLO_HandTools
Program en python que distingue una herramienta de otra;
Bueno, la aplicación que le queremos dar a nuestro programa, es enfocado en resolver una de las problematicas que como mexicanos jovenes que empiezan a laborar en lugares donde se trabaja con herramientas, llegamos a vivir en algun momento, en este tipo de empresas, negocios, talleres, inclusive con padres, etc. Hay ocasiones en las que las personas con mas experencia, al trabajar con personas nuevas, o inexpertas, les hacen bromas pidiendo herramientas inexistentes, o con nombres que suelen ser albueres, como la brida de patas, llaves nudas, entre otras cosas, lo que buscamos resolver con nuestro programa, es crear un programa el cual tenga una base de datos a la cual puedas acuidir a buscar las herramientas que si son existentes, o con los nombres reales de las herramientas, para que de esa manera, las personas que no tengan conocimiento sobre herramientas, puedan de esa manera saber si lo que les estan pidiendo son herramientas reales, o una simple broma de las personas con las que esta laborando.
